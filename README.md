## 📦 项目二：独居生活指南 — 记忆增强型陪伴 Agent

### 项目背景
独居人群面临情感陪伴缺失（78%）、情绪疏导需求（65%）。现有 Agent 多为无状态单次对话，缺乏长期记忆与行为干预能力，用户用完即走。

### 核心能力
- **记忆分层**：短期记忆（会话上下文）+ 长期记忆（用户档案 + 情绪素材库）协同存储
- **正向情绪记录**：每日记录1件开心小事，每月通过 RAG 生成《温情记事薄》PDF
- **行为干预**：上传房间照片 → 视觉分析杂乱程度 → 分步骤清洁指令 → 打勾积分闭环
- **安全边界**：高危关键词识别 + 转人工/转热线兜底流程

### 项目成果
- 覆盖安全守护、情感陪伴、行为引导 3 大场景
- 沉淀「记忆增强 + 行为干预」Agent 设计方法论
- 定义陪伴型 Agent 评估指标：留存率、日记填写率、微任务完成率、eNPS

### 技术方案
- **平台**：Dify（RAG + 工作流）
- **模型**：DeepSeek API + Ollama（本地隐私）
- **向量库**：Chroma
- **视觉**：多模态模型分析房间照片
- **编程**：Python（记忆结构化读写）
